Curriculum by Inference for Verifiable Environments

把 RLVE 的「准确率≥0.9 升难」启发式,换成 active-inference 的行动选择

01问题:固定难度为何「学不动」

GRPO/DAPO 以一组 K 个 rollout 为单位更新。一组全对或全错 → 组内无差异 → 梯度为 0,算力浪费。

一组「有对有错(可用)」的概率:

UK(p) = 1 − pK − (1−p)K　在 p=0.5 最大
  • 太易(p→1)/太难(p→0):UK→0,信号枯竭
  • RLVE 升难点 0.9 → UK(0.9)≈0.57 ≪ UK(0.5)≈0.99:低信号区
02方法:最小化期望自由能 (EFE)

把「选哪个难度训练」当作 active inference 的行动选择:为每个环境维护对「模型能力」的信念 q(s),选 EFE 最小的难度。

Ge(d) = −λsig·UK(p̂) − λinfo·I(s;o|d) + λcost·C(d)
π(d|e) ∝ exp( −Ge(d) / T )
  • pragmatic UK:学习信号(DAPO 有效 prompt 比例)
  • epistemic I(s;o|d):对能力的信息增益
  • T→0 ⇒ 固定 0.5 定点(优于 RLVE 的 0.9);信念不确定 ⇒ 探索,定准 ⇒ 利用(自动)
hidden state s模型能力
action π选难度 d
belief q(s)对能力的后验(IRT 似然 σ(a(s−d)))

06结论
  • 难度 = 一个控制目标(EFE),而非 0.9 阈值
  • 一个温度旋钮统一「利用 / 探索」
  • 主动推断模型能力 → curriculum by inference
  • 边界:模型超过 dmax 时全饱和 → 需环境 scaling(呼应 RLVE)
